md-mouse-hold

AngularJS directive to track mouse click holding

bower install md-mouse-hold --save
Download Development / Production

Setup

Add md.mouseHold to your dependencies

Usage

Click and Hold
The holdingFn(ticks) function is called continuously while the mouse button is being held down.
While the mouse button is held, the element gets the .md-mouse-hold-holding class.

Options

delay Milliseconds between each holding function call (tick) (Default: 5)
onRelease function(duration) called when the mouse button is released (Default: noop)
buttons Array of buttons to listen for ('left', 'middle', right', or ID) (Default: ['left'])
